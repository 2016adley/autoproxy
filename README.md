# autoproxy
shadowsocks, shadowsocksr, jsproxy, tls, https

ssr.sh
脚本说明: ShadowsocksR 真一键安装管理脚本
系统支持: Debian 8+
脚本特点:
伪装https流量

支持 限制 用户速度
支持 限制 端口设备数
支持 显示 当前连接IP
支持 显示 SS/SSR连接+二维码

下载安装:
wget -N --no-check-certificate https://github.com/zqusb/autoproxy/releases/download/1.0/ssr.sh && chmod +x ssr.sh && bash ssr.sh auto
